cv.LogisticRegression/TermCriteria

Termination criteria of the training algorithm.

A struct with the following fields is accepted:

  • type one of 'Count', 'EPS', 'Count+EPS'. default 'Count+EPS'
  • maxCount maximum number of iterations. default Iterations
  • epsilon tolerance value. default LearningRate
