cv.createConcentricSpheresTestSet

Creates test set

[samples, responses] = cv.createConcentricSpheresTestSet(nsamples, nfeatures, nclasses)

Input

  • nsamples returned samples count.
  • nfeatures returned features count.
  • nclasses number of classes.

Output

  • samples returned samples array nsamples-by-nfeatures.
  • sampClasses corresponding classes labels nsamples-by-1.
