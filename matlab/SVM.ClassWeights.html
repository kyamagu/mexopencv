cv.SVM/ClassWeights

Optional weights in the 'C_SVC' problem, assigned to particular classes.

They are multiplied by C so the parameter C of class i becomes ClassWeights(i) * C. Thus these weights affect the misclassification penalty for different classes. The larger weight, the larger penalty on misclassification of data from the corresponding class. Not set by default (empty array []).
