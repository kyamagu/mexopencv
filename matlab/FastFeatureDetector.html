cv.FastFeatureDetector

Wrapping class for feature detection using the FAST method.

Detects corners using the FAST algorithm by [Rosten06].

References:

[Rosten06]:

Edward Rosten and Tom Drummond. "Machine Learning for High-speed Corner Detection". In Computer Vision-ECCV 2006, pages 430-443. Springer, 2006.
