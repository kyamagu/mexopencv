MATLAB File Help: cv.ORB/ScoreType Index
cv.ORB/ScoreType

Algorithm used to rank features.

  • Harris means that Harris algorithm is used to rank features (the score is written to keypoint.response and is used to retain best NFeatures features). This is the default
  • FAST is alternative value of the parameter that produces slightly less stable keypoints, but it is a little faster to compute.
