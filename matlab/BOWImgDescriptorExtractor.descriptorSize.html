MATLAB File Help: cv.BOWImgDescriptorExtractor/descriptorSize Index
cv.BOWImgDescriptorExtractor/descriptorSize

Returns image discriptor size

sz = extractor.descriptorSize()

Output

  • sz Returns an image discriptor size if the vocabulary is set. Otherwise, it returns 0.

This is basically size(Vocabulary,1) (i.e number of clusters).
