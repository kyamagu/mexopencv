cv.KalmanFilter/init

Re-initializes Kalman filter. The previous content is destroyed

kf.init(dynamParams, measureParams)
kf.init(..., 'OptionName', optionValue, ...)

Input

  • dynamParams Dimensionality of the state.
  • measureParams Dimensionality of the measurement.

Options

  • ControlParams Dimensionality of the control vector. default 0
  • Type Type of the created matrices that should be single or double (default).
