MATLAB File Help: cv.LogisticRegression/MiniBatchSize Index
cv.LogisticRegression/MiniBatchSize

Number of training samples taken in each step of Mini-Batch Gradient Descent.

Will only be used if using MiniBatch training algorithm. It has to take values less than the total number of training samples. Default 1
