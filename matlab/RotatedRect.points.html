cv.RotatedRect.points

Returns 4 vertices of the rectangle

pts = cv.RotatedRect.points(rrect)

Input

  • rrect rotated rectangle. A structure with the following fields:
    • center The rectangle mass center [x,y].
    • size Width and height of the rectangle [w,h].
    • angle The rotation angle in a clockwise direction. When the angle is 0, 90, 180, 270 etc., the rectangle becomes an up-right rectangle.

Output

  • pts 4-by-2 points matrix of the rectangle vertices. [x1 y1; x2 y2; x3 y3; x4 y4]
