cv.TwoPassStabilizer/nextFrame

Process next frame from input and return output result

frame = obj.nexFrame()
frame = obj.nexFrame('OptionName',optionValue, ...)

Options

  • FlipChannels in case the output is color image, flips the color order from OpenCV's BGR/BGRA to MATLAB's RGB/RGBA order. default true

Output

  • frame Output result
