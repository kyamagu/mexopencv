MATLAB File Help: cv.HOGDescriptor/getWinSigma Index
cv.HOGDescriptor/getWinSigma

Get window sigma

ws = hog.getWinSigma()

Output

  • ws a numeric value, window sigma.

This returns hog.WinSigma or sum(hog.BlockSize)/8 if it is negative.
